# Contributing to Nexus

We welcome contributions to the Nexus Protocol! Here's how you can help:

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/nexus.git`
3. Install dependencies: `npm install && cd frontend && npm install`
4. Create a branch: `git checkout -b feature/your-feature`

## Making Changes

1. Make your changes
2. Test thoroughly: `clarinet test && npm test`
3. Update documentation if needed
4. Commit with clear messages: `git commit -m "feat: add liquidation mechanism"`

## Pull Request Process

1. Push to your fork: `git push origin feature/your-feature`
2. Open a PR with description of changes
3. Ensure all tests pass
4. Wait for review

## Code Style

- Follow existing patterns
- Add comments for complex logic
- Keep functions focused and small
- Write tests for new features

## Reporting Issues

- Use GitHub Issues
- Provide clear reproduction steps
- Include contract/transaction details if applicable

Thank you for contributing! 🚀
